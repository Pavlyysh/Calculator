# Простой калькулятор на Go

<img width="637" height="361" alt="image" src="https://github.com/user-attachments/assets/f9d406dd-6b73-44ed-b23e-b9ef119c9381" />

Это простое веб-приложение-калькулятор, написанное на языке Go с использованием стандартной библиотеки `net/http` и шаблонизатора `html/template`.

---

## Возможности

- Выполняет базовые арифметические операции:
  - Сложение (`+`)
  - Вычитание (`-`)
  - Умножение (`*`)
  - Деление (`/`)
- Обрабатывает ошибки:
  - Некорректный ввод (не числа)
  - Деление на ноль
  - Неверная операция
- Пользовательский интерфейс в браузере
- Простая и чистая структура проекта

## Запуск

1. Склонируйте репозиторий
2. Запустите приложение `go run main.go`
3. Приложение запустится по адресу `http://localhost:8080`
